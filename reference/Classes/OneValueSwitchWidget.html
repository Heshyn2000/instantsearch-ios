OneValueSwitchWidget

@objc public class OneValueSwitchWidget: SwitchWidget

Widget that controls the facet value of attribute. Built on top of UISwitch.

Note

Use this for one possible value. (Example: premium)

Remark

You must assign a value to the attribute property since the refinement table cannot operate without one. A FatalError will be thrown if you don’t specify anything.
