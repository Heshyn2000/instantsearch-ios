InstantSearch

@objc public class InstantSearch: NSObject, SearcherDelegate

Main class used for interacting with the InstantSearch library. The most important thing that it does is binding the Searcher/Index to the widgets. It also takes care of managing search components: UISearchBar and UISearchController.

    The singleton reference of InstantSearch. It is advised to use this shared reference in case you’re dealing with a single index in your app.

  • register(widget:)

    Register a widget to InstantSearch.

  • param widget: the AlgoliaWidget to be registered to InstantSearch.

    • Note

    if doSearch parameter is not specified, then we will automatically do a search if the AlgoliaWidget is an input control (changes the params of the Searcher), otherwise we don’t do a search.
