RefinementCollectionWidget

@objc public class RefinementCollectionWidget: UICollectionView, RefinementMenuViewDelegate, AlgoliaWidget

Widget that displays facet values for an attribute and lets the user filter the results using these values. Built over a UICollectionView.

Note

This is best used with a RefinementController (through composition) or a RefinementCollectionViewController (through inheritance).

Remark

You must assign a value to the attribute property since the refinement table cannot operate without one. A FatalError will be thrown if you don’t specify anything.
