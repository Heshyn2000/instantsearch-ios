  • MultiHitsTableViewController

    Base class that is meant to be inherited from a ViewController. This class gives you the boilerplate needed to have a multi hits view linked to InstantSearch. It imlpements the tableView dataSource and delegate protocols, and exposes the methods from HitsTableViewDataSource and HitsTableViewDelegate.

    Note

    You have to implement the tableView(_:cellForRowAt:containing:) method in your class to specify the layout of your hit cells, or else you will get a fatalError.
    See more

    Declaration

    Swift

    @objc open class MultiHitsTableViewController: UIViewController,
UITableViewDataSource, UITableViewDelegate, HitsTableViewDataSource, HitsTableViewDelegate

  • MultiHitsController

    A controller object that manages a Hits Widget. It takes care of interacting with InstantSearch and offer clear and easy to use dataSource and delegate methods.

    • tableDataSource: DataSource to specify the layout of a table hit cell.
    • tableDelegate: Delegate to specify the behavior when a table hit cell is selected.
    • collectionDataSource: DataSource to specify the layout of a collection hit cell.
    • collectionDelegate: Delegate to specify the behavior when a collection hit cell is selected.
    See more
